# Startup Co-Founder Agent 

Ever wanted a co-founder who never sleeps, drops startup ideas like a VC, and googles things faster than your intern?  
Meet your new AI buddy: the **Startup Co-Founder Agent**.

## What it does
- Input your startup interest (e.g., "AI for Education")
- Generates 3 startup ideas using **Mistral-7B-Instruct**
- Instantly performs **market research** with Google (via Serper API)
- All in a slick **Streamlit** UI
